Menu de uma padaria: Uma Webpage Personalizada

O projeto consiste em uma página web desenvolvida com HTML e CSS, criada para apresentar o menu de uma padaria como parte de um trabalho escolar. O objetivo é construir um ambiente virtual onde seja possível exibir os principais produtos da padaria, fornecendo uma breve descrição de cada item, incluindo o nome, o preço e uma imagem ilustrativa.

A estrutura do site é baseada em HTML, responsável por definir a organização e o layout do conteúdo, enquanto o CSS é utilizado para estilizar a página, cuidando das cores, da tipografia e da disposição visual dos elementos.

O HTML organiza o conteúdo com um cabeçalho contendo o nome da padaria, seguido por uma seção principal (.container) que apresenta a proposta do site e o cardápio em uma área flexível (.menu). Cada produto é destacado em seu próprio bloco (.item), exibindo uma imagem, as informações sobre o item e o valor. O design responsivo é garantido com o uso da meta tag viewport e de um layout adaptável, que se ajusta a diferentes tamanhos de tela.

O CSS personaliza a aparência do site, utilizando variáveis de cores, fontes importadas do Google Fonts e aplicando um esquema visual agradável, com botões interativos para facilitar a navegação. O uso de HTML e CSS permite criar um site acessível, funcional e atrativo, focado na usabilidade e na boa experiência do usuário. A escolha cuidadosa da tipografia e das cores reforça o aspecto acolhedor e convidativo da padaria, incentivando as pessoas a conhecerem o menu.

Site compra de cursos: Uma nova forma de aprendizagem

O projeto consiste em uma página web desenvolvida com HTML e CSS, criada para apresentar uma plataforma de venda de cursos de estudos como parte de um trabalho escolar. O objetivo é construir um ambiente virtual onde seja possível divulgar diversos cursos, oferecendo informações importantes sobre cada um, como o título, a descrição, o valor e um link para matrícula.

A estrutura do site foi elaborada com HTML, responsável por definir a organização e o layout do conteúdo, enquanto o CSS foi utilizado para estilizar a página, cuidando da harmonia visual através de cores, tipografia e disposição dos elementos.

O HTML organiza o conteúdo com um cabeçalho que apresenta o nome da plataforma, seguido por uma seção principal (.container) onde está a proposta do site e a lista de cursos disponíveis em uma área flexível (.cursos). Cada curso é destacado em seu próprio bloco (.curso), contendo uma imagem ilustrativa, informações detalhadas e um botão que direciona para a matrícula. O design responsivo é garantido pelo uso da meta tag viewport e de um layout que se adapta bem a diferentes tamanhos de tela.

O CSS personaliza o visual do site, utilizando variáveis para padronizar cores, fontes importadas do Google Fonts para uma melhor legibilidade e botões interativos que facilitam a navegação. A escolha do HTML e CSS permite criar um site funcional, acessível e atrativo, com foco na usabilidade e na experiência positiva da pessoa usuária. A seleção das cores e da tipografia contribui para transmitir uma imagem profissional e confiável, incentivando a exploração dos cursos disponíveis.

Contagem regressiva: Uma Aventura até o natal

O projeto consiste em uma página web desenvolvida com HTML, CSS e um pouco de JavaScript, criada para apresentar uma contagem regressiva até o Natal do ano, juntamente com receitas tradicionais natalinas, como parte de um trabalho escolar. O objetivo é construir um ambiente virtual temático, que desperte o espírito natalino e ofereça sugestões de receitas típicas para celebrar a data.

A estrutura do site é organizada com HTML, que define o layout e a hierarquia do conteúdo, enquanto o CSS é responsável pela estilização, aplicando cores, fontes e elementos decorativos relacionados ao Natal. Além disso, o JavaScript é utilizado para implementar a funcionalidade da contagem regressiva, exibindo dinamicamente o tempo restante até o Natal.

O CSS personaliza a aparência do site com um esquema de cores natalinas, como vermelho, verde e dourado, fontes decorativas importadas do Google Fonts, além de botões interativos que incentivam a exploração das receitas. A combinação de HTML, CSS e JavaScript permite criar um site funcional, acessível e visualmente atrativo, focado na experiência da pessoa usuária. A escolha cuidadosa da tipografia e das cores reforça o clima acolhedor e festivo do Natal, tornando o site uma ferramenta divertida e informativa para quem deseja acompanhar a chegada da data e preparar receitas típicas em família.
